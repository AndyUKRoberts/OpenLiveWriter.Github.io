Project Ideas

These are all popularly requested features.

  • Portable version

    This would involve a fork of OpenLiveWriter so that it could run on a thumbdrive or similar device, allowing the user to use someone else's computer without modifying it. See http://portableapps.com/

  • Github Markdown support

    Currently OpenLiveWriter only supports HTML. We would like to support markdown also

  • YouTube Video (and other providers) includes

    A former feature of Live Writer, we need to add support of videos. There are several YouTube videos showing this support

  • Allowing other languages to be used (culturalization)

    Another former feature, requested frequently
